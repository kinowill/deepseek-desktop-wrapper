# Changelog

## 0.4.0 - 2026-04-20

- removed the web wrapper and switched to an API-only desktop architecture
- added collapsible sidebar and tabbed conversations
- added local persistent conversations, drafts, and search
- added multi-profile API key management with encrypted local storage
- added import/export for conversations
- added contextual file attachments
- added custom title bar and frameless window
- added tray icon, splash screen, startup toggle, and notifications
- added controlled logging and log folder shortcut
- added GitHub Releases workflow and auto-update wiring
- added Windows app icon and improved NSIS installer settings
